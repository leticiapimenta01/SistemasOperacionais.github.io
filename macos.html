Histórico

A herança do que viria a ser macOS teve origem na NeXT, uma empresa fundada por Steve Jobs após a sua saída da Apple em 1985. Lá, o sistema operacional Unix-like NeXTSTEP foi desenvolvido e lançado em 1989.

O macOS apresenta uma grande mudança em relação aos sistemas operacionais anteriores do Macintosh e o código que é a base do sistema é completamente diferente das versões anteriores. Apesar da maioria das alterações estarem nos "bastidores" do sistema operacional, a interface gráfica de usuário Aqua é o novo recurso mais visível.O núcleo do macOS é um sistema operacional certificado Unix, construído em torno de um núcleo XNU com recursos básicos Unix acessíveis por meio de uma interface de linha de comando.

O Mac OS X foi originalmente apresentado como a décima versão principal do sistema operacional da Apple para computadores Macintosh; as versões atuais do macOS mantêm o número de versão principal "10". Os sistemas operacionais Macintosh anteriores (versões do Mac OS clássico) eram nomeados usando números arábicos, como no Mac OS 8 e Mac OS 9.

A letra "X" no nome do Mac OS X refere-se ao número 10 no numeral romano. É, portanto, corretamente pronunciada como "dez" nesse contexto. No entanto, é também comumente pronunciado como "X" (xis).

Principais Características

O OS X apresenta uma grande mudança comparadas aos sistemas operacionais anteriores do Macintosh com os códigos de linguagens, base do sistema, cujo se torna completamente diferente das versões anteriores. Além da maioria das alterações estarem entre o sistema operacional, a interface gráfica de usuário Aqua é o novo recurso mais visível.Com uso de bordas arredondadas, cores translúcidas e outros adereços trouxe mais cores e texturas às janelas e controles do Desktop do que a interface Platinum do OS.

Você pode abrir vários aplicativos ao mesmo tempo sem dar aquelas complicações de travamento ou lentidão no PC, pois além disso esse sistema operacional obtêm formas de seguranças mais acessíveis para aqueles que usam o MAC entender o que realmente quer na proteção de seu computador.

O MAC OS X é totalmente objetivo nas suas funções para que os usuários tenham a facilidade de aprender a mexer no sistema operacional.

Por defeito, o Mac OS X apenas nos dá acesso de administrador e não de root, trata-se de um esquema de prevenção para que nenhum usuário danifique o sistema de tal forma que o torne inutilizável.

Requisitos Mínimos para Instalação

O Mac OS X versão 10.5 Leopard requer:

Um computador Mac com processador Intel, PowerPC G5 ou Power PC G4 (867 MHz ou superior)

Memória de 512 MB ou mais

Uma unidade de DVD para instalação

9 GB ou mais de espaço livre disponível em disco.
